# DRONES-3D-ANEM-2025

Simulation 3D temps réel de spectacle de drones pour la cérémonie d'ouverture ANEM 2025.

## Installation

```bash
pip install -r requirements.txt
```

## Structure
- `src/`: Code source
- `data/`: Données de formation et assets
- `config/`: Fichiers de configuration
- `outputs/`: Fichiers générés (logs, vidéos)

## Lancement

```bash
python src/main.py
```
